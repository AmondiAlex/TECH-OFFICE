involves functional checks, servicing,

Hardware

Hardware refers to the computer's tangible
components or delivery systems that store and
run the written instructions
provided by the software.

Software

Software is a set of instructions, data
or programs used to operate computers
and execute specific tasks.
It is the opposite of hardware,
describes the physical aspects of a computer.
